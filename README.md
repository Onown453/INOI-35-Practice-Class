# INOI-35-Practice-Class

بعضی سوالات تست کیسشون سنگین بود لود نشد
برای ساخت تست کیس به فولدر سوال برید
بعد
`./gen.sh 20`
بیست تا تست کیس می‌سازه.

سپس دوباره به فولدر اولیه برگردید و با کمک
`./run_test.sh name_of_problem/`
تست کنید که تست کیسا ولید هستن یا نه.

به طور خلاصه

```
cd envy/
./gen.sh 20
cd ..
./run_test.sh envy
```
